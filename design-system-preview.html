Type

Editorial serif titles, Outfit for everything else. os.me's proven model.

Bodoni Moda · display optical size · titles only

Monish Kamwal
Draw something.

A true didone: high-contrast, hairline strokes, near-circular bowls, tall caps. Set all-caps with negative tracking (~−.02em) at display sizes; the highest optical size (opsz 96) gives the thinnest hairlines. No swash alternates — the contrast carries the character.

Reference sheet only — the design in parts, not the built site. Source of truth is DESIGN.md; implementation waits on the mlops build (Stage 5).

Contrast rules shown live: bright ochre #c98a3a does every large job (arrow, bar fills, hovers); small text & links use deep ochre #96600a for AA. Drawn lines & dots are ink #0a0a0a; the hairline #e8e8e8 is white-surfaces only.

Typefaces load from Google Fonts — open with an internet connection to see the real Bodoni Moda hairlines; otherwise it falls back to Georgia / system sans.
